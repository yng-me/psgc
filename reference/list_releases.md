# List available PSGC releases

List available PSGC releases

## Usage

``` r
list_releases()
```

## Value

A character vector of release names in chronological order.

## Examples

``` r
list_releases()
#>  [1] "Q1_2023"    "Q4_2023"    "April_2024" "Q2_2024"    "Q3_2024"   
#>  [6] "Q4_2024"    "Q1_2025"    "Q2_2025"    "July_2025"  "Q3_2025"   
#> [11] "Q4_2025"    "Q1_2026"   
```
